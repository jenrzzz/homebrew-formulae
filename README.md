# Jenrzzz Formulae

## How do I install these formulae?

`brew install jenrzzz/formulae/<formula>`

Or `brew tap jenrzzz/formulae` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jenrzzz/formulae"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
